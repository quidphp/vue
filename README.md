## About
**QuidPHP/Vue** repository contains a sample application project built on top of the QuidPHP framework. This application is using LemurCms for content management.

## License
**QuidPHP/Vue** is available as an open-source software under the [MIT license](LICENSE).

## Documentation
**QuidPHP/Vue** documentation is being written. Once ready, it will be available at [QuidPHP/Docs](https://github.com/quidphp/docs).

## Installation
**QuidPHP/Vue** can be easily installed with [Composer](https://getcomposer.org). It is available on [Packagist](https://packagist.org/packages/quidphp/vue).
``` bash
$ composer create-project quidphp/vue --prefer-dist
```
